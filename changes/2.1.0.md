### Seata-go 2.1.0

Seata-go 2.1.0 Released.

Seata-go is an easy-to-use, high-performance, open source distributed transaction solution.

The version is updated as follows:

### feature：

- [[#868](https://github.com/apache/incubator-seata-go/pull/868)] Add script manager&actuator
- [[#871](https://github.com/apache/incubator-seata-go/pull/871)] Support Saga mode config initialization
- [[#897](https://github.com/apache/incubator-seata-go/pull/897)] Saga end-to-end verification
